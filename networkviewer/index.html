Complex Networks - WebGL Viewer

Assignment for Computational Graphics graduate course


ICMC - Universidade de São Paulo

Filipi Nascimento Silva


Interactive Network Visualization - BETA
The software requires a WebGL enabled browser.
Check for the latest versions of Chrome, Firefox or Safari(Enable WebGL in Develop Menu).
http://get.webgl.org/

This software uses WebGL to generate and display networks visualizations together with several options of properties.


Basic Controls:
  • Left-Mouse button dragging: Rotates the network
  • Mouse Wheel: Zoom In/Zoom Out
